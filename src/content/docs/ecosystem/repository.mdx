Browse and install all available plugins directly from the DVGE application. Copy a plugin's ID and use it in the **Catalog** section of the app.

## How to Install a Plugin
1. Open DVGE.
2. Go to the **Catalog** tab in the main menu.
3. Find the plugin by name or paste its ID.
4. Click **Install** — the engine handles the rest.
